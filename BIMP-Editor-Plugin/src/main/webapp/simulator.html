Online Simulator

Here are some instructions


  1. Please select a valid BPMN or Microsoft Visio 2013 BPMN (vsdx, beta support) file.
  2. Press "Continue" in order to add/change simulation information.
  3. Tick the "Generate a log" box if you want to be able to download simulation MXML log afterwards.
  4. Click "Start simulation".
  5. Be amazed and wonder how such magic came to be.

You can find a sample process model to try out here. Just unzip and upload the BPMN model!
